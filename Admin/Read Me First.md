# Read Me First

**Project:** Detection-Rule Leakage and Trust-Camouflage  
**Author:** Michael Zot  
**ORCID:** 0009-0001-9194-938X

This archive contains the public research materials for **Detection-Rule Leakage and Trust-Camouflage**, a function-over-form model of adversarial adaptation in open-audience mental-health communication.

The OSF archive is the canonical peer-review package. The GitHub repository is a public materials mirror. The Zenodo record archives the GitHub release for repository preservation and indexing.

## Contents

- Scholarly manuscript and LaTeX source
- Trust Camouflage Function Score research-use coding materials
- H2 preregistration materials and vignette stimuli
- H2 Qualtrics, jsPsych, Prolific, example export, and analysis materials
- TC LaunderBench benchmark materials for LLM rhetorical laundering tests
- Literature-gap search materials
- Validation roadmap and next-study protocols
- Manifest files with file sizes and SHA-256 checksums

## Scope and use

These materials are for research use, peer review, and controlled empirical testing. The coding protocol is not a clinical instrument, diagnostic tool, custody tool, HR tool, or relationship verdict. The framework evaluates observable function over time: repair, evidence, accountability, consequences, and repeated pattern.

# MIPU Reversal Cost Experiment – Protocol

## Purpose

Quantify whether a minimal trust-bearing cue shifts accountability judgments.

## Design

Within-subjects pre/post, N ≈ 80.

## Procedure

1. Present baseline conflict vignette.
2. Measure blame assignment, accountability obligation, repair urgency, credibility.
3. Inject one MIPU cue.
4. Re-measure outcomes.

## Primary outcome

Delta Accountability = Post-MIPU Accountability Composite – Pre-MIPU Accountability Composite.

This is the measured burden shift / reversal cost.

## Analysis

Paired t-test, Cohen's d, 95% CI.
